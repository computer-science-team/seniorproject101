Manual For Faculty Users

As a Faculty User, you may find some of this information useful...


On sign-up:

Select the faculty member option from the drop down box to create the faculty experience.

After sign-up:

As a faculty account was created, a faculty member will belong to a specific university. Faculty members have the ability to create visible or invisible tools to the related university toolbox for student users to have access. This is protect tools needing to be seen or are yet to be released to the public. Guidelines of success, list of clubs, and IDE's are supplied by the faculty user as PDF files.
